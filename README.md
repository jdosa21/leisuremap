# Leisure Map Project


## 프로젝트개요 
### * 프로젝트 개발 기간: 2022.10.22 - 2022.11.21
### * 프로젝트 개발 배경: 코로나19가 장기화되면서 해외여행의 어려움과 실내 활동의 제약을 극복하기 위해 혼자서 하는 여가활동과 실외공간에서 하는 여가활동의
### 인식이 증가하였습니다. 이와 같은 트렌드에 맞춰 더 나은 정보와 서비스를 제공하기 위해 기획했습니다.
### * 개발 요소 : Java, Spring Boot, MariaDB, HTML, CSS
